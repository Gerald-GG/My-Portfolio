Projects

  1. Face Recognition Attendance system

    I developed this project using python programming language.
    This project was inspired by a biometric system that I observed in Pioneer University.
    The biometric system records and stores collected students data.
    This data included time-stamps for students recorded as they entered the lecturehalls.
    I developed The Face Recognition System that did not only record students time attendance stamps,
    but also records the total attendance for each student.
    The overall attendance for the students per lecture-hall is recorded and stored in excell sheets.
    These daily excellsheets are sent attomatically to the lecturer, the Chair of Department,
    Deen of Students and Registrar.

  2. Weather Prediction App

    I builded this app to gather real-time weather data, processing it and presenting it in a user friendly way.
    I used an API that was reliable for collecting data.
    The data included, humidity, temperature, wind speed, rain and more.
    I developed the app to be location based forecast, meaning, that it provided weather forecast for user's current location,
    and it allowed users to search for weather information in other locations.
